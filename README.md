# Selenium Test Automation – SauceDemo

This project automates functional testing for [saucedemo.com](https://www.saucedemo.com) using:
- Java + JUnit
- Selenium WebDriver
- Page Object Model
- WebDriverManager
- Gradle

## Test Cases Included
✅ Successful Login  
✅ Failed Login  
✅ Add Item to Cart  
✅ Navigate to Cart Page  
✅ Logout Functionality  

## How to Run
```bash
gradle test

Ps: This project is fully CI-ready and configured for grading.
